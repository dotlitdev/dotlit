    • Links

    Links in .lit are broken down into two main types StandardÂ§ and wiki-links. mich of the complexity is a result of the second type.

    Regular links

    Should for the most part behave as expected, with the recommendation being that you should link to the source for other .lit/Mardown documents and the resultant html should convert to .html extention automatically.

    WikiLinks

    These are a convention introduced by Wiki and rely on a few assumptions.

    1. There is a canonical item to link to
    2. Or a system for handling disambiguation
    js
    const { wikiLinkOptions, linkToUrl, decorateLinkNode } = lit.parser.utils;

const resolve = wikiLinkOptions().pageResolver;

// console.info(resolve.toString())
// console.info(linkToUrl.toString())

const root = "/";
const type = "wikiLink";
const example = "My cool ðŸ˜Ž link";

  url: example,
  value: example,
};

console.log({ root, example, resolved, link , url});

return JSON.stringify(decorated, null, 2);
